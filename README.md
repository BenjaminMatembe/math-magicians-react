### Prerequisites

In order to run this project you need:


Clone the code from project using terminal into your local machine and `npm install` to install node modules, then `npm start` to run the project in your browser.

### Setup


>Clone this repository to your desired folder using:

 <h1>Setup SSH-key and use this command</h1>
 git clone git@github.com:BenjaminMatembe/math-magicians-react.git

>running npm install or yarn install
